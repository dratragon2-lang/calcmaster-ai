# CalcMaster AI

Sistema inteligente para el cálculo, análisis y visualización de operaciones matemáticas avanzadas.

## Tecnologías

- React + Vite
- Node.js
- PostgreSQL
- Framer Motion
- KaTeX
- TailwindCSS

## Características

- Cálculo de derivadas
- Integrales
- Visualización matemática
- Historial persistente
- Login y autenticación
- Dashboard SaaS moderno
- Exportación PDF
- Gráficas matemáticas

## Instalación

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Base de Datos

PostgreSQL

Configurar archivo:

```env
backend/.env
```
